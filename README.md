# Startup Manager

Startup Manager is a simple desktop utility for viewing and managing
applications that start automatically with Windows.

No background services. No hidden behavior.

## Features
- View startup applications
- Enable or disable startup entries
- No telemetry
- Lightweight and fast
- Portable

## Design principles
- Full user control
- No automation without consent
- Simple and clear UI

## Status
Beta

## Philosophy
Independent software. Privacy by design.
